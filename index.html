how to use

✦ a quick walkthrough

What this is

A helper that writes your {{user}} replies for Crushon.AI (or any roleplay app). You paste what the character said, tell it what you want to do, and copy the reply back into Crushon.

First-time setup

1 Get a free API key at openrouter.ai/keys (sign in with Google). Paste it in ⚙ API Key & Model.

2 Tap 👤 Me (top right) → fill in your character's name, appearance, outfit, personality.

3 Tap ☰ Chats → + New → name the character you're roleplaying with.

Each turn

A Copy what {{char}} just said in Crushon → paste in the green box.

B Write a quick direction in the rose box — what you want to do/say.

C Tap Generate → → wait → tap 📋 copy on the reply → paste into Crushon.

NSFW

To allow NSFW, two switches: turn it on per character (in Me drawer), then tap the NSFW pill above the Generate button.

Tips

· Tap ↻ regen if you don't like a reply — swipe between versions with ‹ 1/2 ›.

· Tap ✎ edit to fix small things before copying.

· If "Load failed" — open this site from your hosted URL (like GitHub Pages), not as a file.
